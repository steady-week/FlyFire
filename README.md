# 飞火商场

参考苏宁，京东，淘宝的设计样式，准备自己撸一个电商网站，使用HTML，CSS，JavaScript，font awesome，jQuery。


开发环境采用Windows。
发开工具采用webstorm,git，nodeJS。
打包工具使用webpack。
具体对HTML，CSS，图片，JavaScript打包工具见package.json。
